# Lab Sheet 05: Vehicle Rental & Delivery Charge Calculator

A Java-based Object-Oriented Programming (OOP) implementation that simulates a vehicle rental management system and a dynamic delivery charge calculator. This project demonstrates core OOP principles such as inheritance, polymorphism, and encapsulation.

---

## 📂 Project Structure

The repository contains the following core Java files:

*   **`Vehicle.java`** – The base (parent) class defining common attributes and behaviors for all rental vehicles.
*   **`Car.java`** – A subclass inheriting from `Vehicle`, tailored for car-specific attributes and rental calculations.
*   **`Bike.java`** – A subclass inheriting from `Vehicle`, tailored for motorbike/bicycle-specific attributes and rental calculations.
*   **`DeliveryChargeCalculator.java`** – A utility class that calculates delivery rates based on vehicle types, distance, or predefined business logic.
*   **`VehicleRentalTest.java`** – Test suite/main execution entry point to verify rental mechanics, vehicle instantiation, and polymorphic behavior.
*   **`DeliveryChargeTest.java`** – Test suite/main execution entry point to validate delivery charge computations across different edge cases.
